# Практика
# Задание «Вариант 1. Игра «Захват объектов»»

Пользователь управляет квадратом-персонажем, который двигается по игровому полю с помощью стрелок. На поле случайным образом появляются объекты, которые нужно поймать соприкосновением. Каждый захваченный объект добавляет одно очко к счету. Конец игры по окончании времени, таймер.


# Установка и использование

1. **Visual Studio** последней версии (в моём случае установлена VS 2026)
2. Установлен пакет зависимостей - **Разработка классических приложений .NET**
